# pro-tnt-bot

Реализация полноценного телеграм бота (бета версия).

## Переменные окружения
  - `BOT_TOKEN` - Токен бота.
  - `BOT_CREATOR_ID` - Идентификатор создателя.
  - `BOT_USERNAME` - Юзернейм бота, без `@`.
